### Install:
1. `mkdir <your_root_project_folder>`  
2. `cd <your_root_project_folder>`
3. `curl -L https://api.github.com/repos/JackRabbit911/Docker_php8.2/tarball/master > docker.tar`
4. `tar -xf docker.tar`
5. `mv JackRabbit911-Docker_php8.2*/{*,.[^.]*} .`
6. `rm docker.tar`
